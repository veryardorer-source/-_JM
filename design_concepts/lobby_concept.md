# 회사 로비 디자인 컨셉

## 고객 요구사항
- 식물 배치
- 고객 대기 의자
- 한 면: 자사 제품 공간
- 입구 정면: 고객은 LED패널 희망 → 사이니지로 제안
- 천장: 루버 (고객사 레퍼런스)
- 바닥: 데코타일
- 저예산
- 카페테리아 포함

## 컨셉 방향
**"따뜻한 기업 쇼룸형 로비"**
- 키워드: 웰커밍 / 바이오필릭 / 프로덕트 쇼케이스 / 저예산 브랜딩

## 구역별 계획

| 구역 | 방향 |
|---|---|
| 정면(입구) | 백릿 사이니지 + 로고 월 (LED 대신) |
| 제품면 | 오픈 선반 + 스팟조명, 우드/화이트 마감 |
| 천장 | 우드 루버 부분 적용 (대기석·카페테리아 위) |
| 대기 의자 | 벤치형 + 쿠션 |
| 식물 | 대형 1 + 중형 2-3 (극락조, 몬스테라, 산세베리아) |
| 카페테리아 | 바 테이블 + 하이체어, 오픈형 |
| 바닥 | **테라조 데코타일 (흰색/베이지 바탕 + 그레이 돌조각)** |

## 저예산 핵심 전략
1. 루버·우드는 부분 포인트만 (전체 시공 X)
2. 사이니지는 간접조명으로 입체감 연출
3. 벤치·바테이블 같은 다목적 가구로 품목 수 축소
4. 식물로 공간감 보완 (마감재보다 가성비 높음)

## 사이니지 vs LED 설득 포인트
- LED패널: 설치비 외 영상 콘텐츠 지속 제작·교체 부담, 고장 시 부분 수리 어려움
- 사이니지: 입체 로고 + 간접조명으로 브랜드 정체성 고정
- 같은 예산으로 카페테리아·제품 쇼케이스에 투자 여력 확보

## 다음 결정 사항
1. 사이니지 마감: 백릿(간접조명) / 무광 메탈 / 우드 입체
2. 대기 벤치 색상: 따뜻한 패브릭(베이지·카멜) / 차분한(차콜·그레이)
3. 제품 쇼케이스 벽면: 화이트 우드 / 원목 톤 / 도장
4. 루버 우드톤: 밝은 오크 / 중간톤 월넛
